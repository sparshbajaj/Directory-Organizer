# Design System (TUI)

## Visual Language
- Clean, high-contrast terminal UI.
- Emphasis on readability and status clarity.

## Color Tokens (Theme Names)
- Primary: cyan
- Accent: green
- Warning: yellow
- Error: red
- Muted: gray

## Typography
- Terminal default monospace.
- Emphasize with bold and inverse for focus.

## Layout
- Two-pane layout: list on left, detail on right.
- Top status bar with progress and mode indicators.

## Components
- Button: [Label]
- Toggle: [On/Off]
- Input: [________]
- Badge: {AI}, {Manual}, {No-Change}
- Status: OK / WARN / ERROR

## Interaction Rules
- Keyboard-first navigation.
- Arrow keys move selection.
- Enter opens detail.
- A approves, S skips, E edits.

## Motion
- Minimal terminal animations (spinner only).
