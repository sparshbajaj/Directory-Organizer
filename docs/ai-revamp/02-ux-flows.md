# UX Flows

## Flow 1: First Run
1. Launch TUI.
2. Select target folder (default Downloads).
3. Confirm AI content analysis notice.
4. Start scan.
5. View results list with filters.
6. Inspect a suggestion detail (rationale, confidence).
7. Approve all or select items.
8. Apply changes.
9. View summary and save run history.

## Flow 2: Review Suggestions
1. Open results list.
2. Filter by type (rename, move, no-change).
3. Sort by confidence or risk.
4. View detail and edit suggestion if needed.
5. Mark approve/skip.

## Flow 3: Map View
1. Switch to Map.
2. Zoom cluster (project/topic).
3. Highlight related files.
4. Jump to suggestion list filtered by cluster.

## Flow 4: Undo
1. Open history panel.
2. Select last run.
3. Review operations.
4. Undo and verify.

## Flow 5: Dry Run
1. Enable dry-run.
2. Run scan.
3. Export preview file.
4. Exit without changes.
