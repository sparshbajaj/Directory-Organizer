# MVP Scope

## In Scope
- TUI shell with scan configuration.
- Recursive scan and file list.
- AI analysis (default enabled) with safe gating.
- Rename/move suggestions with confidence.
- Review list + detail pane.
- Dry-run export.
- Apply changes with undo.

## Out of Scope
- Web UI.
- Multi-user or cloud sync.
- Full-text indexing across drives.
